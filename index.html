jQuery permite agregar efectos a un sitio web de forma sencilla. Cada uno de los efectos poseen una configuración por defecto, sin embargo, es posible editar los parámetros personalizados para poder realizar diseños a medida. 

$(document).ready(function(){
    $("#hide").on('click', function() {
        $("#element").hide();
        return false;
    });
 
    $("#show").on('click', function() {
        $("#element").show();
        return false;
    });
});

Hide and Show

jQuery permite mostrar y ocultar elementos de HTML utilizando los métodos show() y hide().

Sintaxis: $("p").show();

Sintaxis: $("p").hide();

Ejemplo 1 show y hide

$(document).ready(function(){
    $(".hide-btn").click(function(){
        $(".e2").hide();
    });
    
    $(".show-btn").click(function(){
        $(".e3").show();
    });
});

Toggle

Toggle es un método de jQuery que permite alternar entre mostrar y ocultar un elemento HTML.

Sintaxis: $("p").toggle();

Ejemplo

$(document).ready(function(){
  $(".toggle-btn").click(function(){
    $("e4").toggle();
  });
});

Fade

Fade es un efecto de animación que permite dismininuir o aumentar la opacidad de un elemento.

Existen cuatro tipos de Fade en jQuery: fadeIn, fadeOut, fadeToggle, y fadeTo

FadeIn permite fundir un elemento desde la opacidad mínima hasta la máxima opacidad.

Sintaxis: $("p").fadeIn( [velocidad, callback] );

FadeOut permite fundir un elemento desde la opacidad máxima hasta la mínima opacidad.

Sintaxis: $("p").fadeOut( [velocidad, callback] );

FadeToggle permite fundir un elemento desde la opacidad máxima hasta la mínima opacidad, y luego alternar fundiendo el elemento desde la opacidad mínima hasta la máxima opacidad, o viceversa.

Sintaxis: $("p").fadeToggle( [velocidad, callback] );

FadeTo permite fundir un elemento hasta una opacidad determinada.

Sintaxis: $("p").fadeTo( [velocidad, opacidad, callback] );

FadeIn, FadeOut, FadeToggle, y FadeTo permiten agregar parámetros adicionales que modifican su compartamiento. La velocidad permite definir qué tan rápido será la animación, medida en milisegundos, o utilizando los valores de "slow", "normal" o "fast". Callback es un parámetro opcional para volver a llamar la función una vez ha terminado una operación completa.

Ejemplo 1

$(document).ready(function() {
    $("#mostrar5").click(function(event) {
        $("#rectangulo").fadeIn();
    });
    $("#mostrar6").click(function(event) {
        $("#rectangulo").fadeIn(4000);
    });
    $("#mostrar7").click(function(event) {
        $("#rectangulo").fadeIn(4000, function() {
            alert ('Efecto fadeIn terminado!');
            });
    });
    $("#ocultar5").click(function(event) {
        $("#rectangulo").fadeOut();
    });
    $("#ocultar6").click(function(event) {
        $("#rectangulo").fadeOut(4000);
    });
    $("#ocultar7").click(function(event) {
        $("#rectangulo").fadeOut(4000, function() {
            alert ('Efecto fadeOut terminado!');
            });
    });
});

Slide

Slide es un efecto que realiza un desplazamiento de un elemento en alguna dirección.

Existen tres tipos de Slide en jQuery: slideDown, slideUp, slideToggle

slideDown permite deslizar un elemento hacia abajo.

Sintaxis: $("p").slideDown( [velocidad, callback] );

slideUp permite deslizar un elemento hacia arriba.

Sintaxis: $("p").slideUp( [velocidad, callback] );

slideToggle permite alternar entre deslizar un elemento hacia abajo y deslizar un elemento hacia arriba.

Sintaxis: $("p").slideToggle( [velocidad, callback] );

slideDown, slideUp, slideToggle permiten agregar parámetros adicionales que modifican su compartamiento. La velocidad permite definir qué tan rápido será la animación, medida en milisegundos, o utilizando los valores de "slow", "normal" o "fast". Callback es un parámetro opcional para volver a llamar la función una vez ha terminado una operación completa.

Ejemplo 1

$(document).ready(function() {
    $("#down01").click(function(event) {
        $("#rectangulo3").slideDown();
    });
    $("#down02").click(function(event) {
        $("#rectangulo3").slideDown(4000);
    });
    $("#down03").click(function(event) {
        $("#rectangulo3").slideDown("fast");
    });
    $("#down04").click(function(event) {
        $("#rectangulo").slideDown("fast", function() {
            alert ('Efecto slideDown con velocidad fast terminado!');
        });
    });

    $("#up01").click(function(event) {
        $("#rectangulo3").slideUp();
    });
    $("#up02").click(function(event) {
        $("#rectangulo3").slideUp(4000);
    });
    $("#up03").click(function(event) {
        $("#rectangulo3").slideUp("slow");
    });
    $("#up04").click(function(event) {
        $("#rectangulo3").slideUp("slow", function() {
            alert ('Efecto slideUp con velocidad slow terminado!');
        });
    });

    $("#toggle01").click(function(event) {
        $("#rectangulo3").slideToggle();
    });
    $("#toggle02").click(function(event) {
        $("#rectangulo3").slideToggle(4000);
    });
    $("#toggle03").click(function(event) {
        $("#rectangulo").slideToggle("slow", function() {
            alert ('Efecto slideToggle con velocidad slow terminado!');
        });
    });
});

Animate

Con animate se puede crear efectos de animación modificando propiedades propias de CSS que pertenecen a los elementos del DOM de un sitio web.

Sintaxis: $(elemento).animate(propiedadesCSS,duracion,easing,callback)

Ejemplo 1

$(document).ready(function(){
    $("#boton1").click(function(){
        $("#paraanimar").animate({padding:"0px"});
    });
    $("#boton2").click(function(){
        $("#paraanimar").animate({padding:"100px"});
    });
    $("#boton3").click(function(){
        $("#paraanimar").animate({paddingBottom:"150px"},3000);
    });
    $("#boton4").click(function(){
        $("#paraanimar").animate({paddingTop:"150px"},5000,function() {alert ('Efecto terminado!');});
    });
    $("#boton5").click(function(){
        $("#paraanimar").animate({paddingLeft:"150px", paddingRight:"150px"});
    });
});

stop

Stop es un método de jQuery que permite parar una animación o algún efecto una vez han empezado. En caso de que, por ejemplo, se llamó al efecto slideUp para desplazar un elemento hacia arriba, y se acciona el método stop, el elemento seguirá mostrándose, pero en la posición exacta dónde finalizó la animación sin reiniciarla.

Sintaxis: $("p").stop(stopAll,goToEnd);

Ejemplo 1

$(document).ready(function()
    {
    $("#comenzarefectos").click(function(){
        $("#funciones").slideUp(6000);
        $("#funciones").slideDown(6000);
    });
 
    $("#paratodo").click(function(){
        $("#funciones").stop(true);
    });
 
    $("#finalizaefectoactual").click(function(){
        $("#funciones").stop(false,true);
    });
});


$(document).ready(function()
    {
    var MiColaDeFunciones = $("#funciones");
 
    $("#botoncomenzarcola").click(function(){
            MiColaDeFunciones.fadeOut(6000);
            MiColaDeFunciones.fadeIn(6000);
    });
 
    $("#botonstopcola").click(function(){
        MiColaDeFunciones.stop();
    });
});

Callback

Las declaraciones de JavaScript se ejecutan línea por línea. Sin embargo, con los efectos, se puede ejecutar la siguiente línea de código aunque el efecto no haya finalizado. Esto puede crear errores. Para evitar esto, puede crear una función de devolución de llamada. Una función callback de llamada se ejecuta después de que finaliza el efecto actual.

Sintaxis: $("p").efecto(speed, callback)

Ejemplo 1

$(document).ready(function() {
    $("#down05").click(function(event) {
        $("#rectangulo").slideDown("fast", function() {
            alert ('Efecto slideDown con velocidad fast terminado!');
        });
    });
    $("#up05").click(function(event) {
        $("#rectangulo").slideUp("slow", function() {
            alert ('Efecto slideUp con velocidad slow terminado!');
        });
    });
    $("#toggle05").click(function(event) {
        $("#rectangulo").slideToggle("slow", function() {
            alert ('Efecto slideToggle con velocidad slow terminado!');
        });
    });
});

Chaining

Chaining es la técnica de aplicar una secuencia de efectos de jQuery, ejecutados en secuencia a un objeto del DOM de HTML.

Sintaxis: $("p").efecto1().efecto2().efecto3();

Ejemplo 1

$(document).ready(function(){
    $("#parrafo").hide(4000).show(4000).fadeOut(4000).fadeIn(4000).slideUp(4000).slideDown(4000);
});
